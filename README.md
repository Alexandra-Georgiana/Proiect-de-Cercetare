# AI Art Research Project

**Course:** Proiect de Cercetare  
**Faculty:** Faculty of Mathematics and Computer Science, Babeș-Bolyai University, Cluj, Romania  
**Author:** Alexandra-Georgiana Alistar  

## Description
This repository contains the research project for the course **Proiect de Cercetare**, investigating emergent style, emotional fidelity, and originality in AI-generated art. The project combines computational analysis and human perception studies to evaluate AI artworks.

## Contents
- **Data/**: AI-generated and reference human artworks
- **Notebooks/**: Code for feature extraction, embeddings, and analysis
- **Figures/**: Visualization of results (UMAP, clustering, etc.)
- **Report/**: Full research report including methodology, experiments, and results
- **References/**: Bibliography and citations

## Methodology
- Feature extraction using **CLIP** and **DINOv2** embeddings
- Texture and material analysis (STS, MCA, LMC)
- Emotional fidelity evaluation via computational metrics
- Originality assessment with perceptual hashing and dataset similarity
- Emergent Style Index computation and correlation analysis

## Requirements
- Python 3.10+
- Packages: `numpy`, `pandas`, `scikit-learn`, `umap-learn`, `matplotlib`, `torch`, `transformers`
